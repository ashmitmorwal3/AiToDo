# **App Name**: TaskTango

## Core Features:

- Task Creation: Add tasks to a list with a clear and simple input field.
- Task Completion: Mark tasks as complete with a checkbox or similar interactive element.
- Task Display: Display all tasks with a visual distinction between pending and completed tasks.
- Task Deletion: Allow users to delete tasks from the list.
- AI-Powered Deadline Suggestions: AI-powered feature that suggests a reasonable deadline for a given task. The AI model acts as a tool to suggest dates; the user can either accept or reject the suggestion.

## Style Guidelines:

- Primary color: A calming blue (#3498db) for headers and main elements.
- Secondary color: A light gray (#ecf0f1) for backgrounds and less important elements.
- Accent: A vibrant green (#2ecc71) to indicate task completion and success.
- Clean and readable sans-serif font for task descriptions.
- Simple and clear icons for actions like adding, deleting, and completing tasks.
- Clean and well-spaced layout to ensure readability and ease of use.
- Subtle animations for task completion and deletion to provide visual feedback.
